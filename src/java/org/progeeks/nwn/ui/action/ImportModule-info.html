This page lets you configure the general project information like the project name, project description, and output module file.


Project Name

This is the name of the project and the one that will be used to create the project's XML file in the directory specified when creating the project.


Target Module Name

This is the name of the module file that will be written when doing a full module build.
Project Description

This is a general description for the project. It defaults to being the same as the module description and if the module does not contain a description, this one will be used in its place. However, the two descriptions can be different if desired.
