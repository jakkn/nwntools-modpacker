This page lets you configure the sub-directories that the project will use. All directories that are sub-directories of the project directory will be treated as relative.

Project directory: [@directory@]


Build Directory Defaults to "./build"

This is the directory where Pandora will build all intermediate binaries before packaging them into a module.
Work Directory Defaults to "./cache"

This is the directory where Pandora stores temporary working files for the project. This may include cached images, dependency graphs, etc..
Source Directory Defaults to "./source"

This is where the actual user-modifiable data files are stored. If storing the project in a source control system, you only need to store this directory and the project file.
