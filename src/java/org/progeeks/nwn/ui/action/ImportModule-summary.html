Project: @project@

This page details the operations the will be performed to create the project [@project@] from the imported module [@module@].

Operations:

  • Create project file: [@project-file@]
  • Create project directories:
    • @build-directory@
    • @work-directory@
    • @source-directory@
  • Unpack [@module-file@] into directory: [@build-directory@]
  • Convert module files and store in: [@source-directory@]

Press "Finish" to perform the above operations, or "Cancel" to abort
